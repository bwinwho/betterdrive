BetterDrive

One-time setup · Google Cloud keys

Get your keys — 5 minutes, free

  1. Open console.cloud.google.com → create a project named BetterDrive.
  2. In APIs & Services → Library, enable Google Drive API and Google Picker API.
  3. In OAuth consent screen: choose External, fill app name, add your email under Test users.
  4. In Credentials → Create credentials → OAuth client ID → type Web application. Under Authorized JavaScript origins add the exact URL you'll open this app from, e.g. https://yourapp.pages.dev (and http://localhost:8000 for testing).
  5. Copy the Client ID. Then Create credentials → API key, copy it too.
